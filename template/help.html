Help: Posts

A post represents a uploaded file that can be commented on and searched for. Alternatively if you upload a post of your own you can give it original poster and source attributes. More about uploading read Uploading section, as well as caption and add tags that can be easily searched for as before mentioned. More about tags read Tags section

Help: Search

You can search for tag game gtav and it will search for Grand Theft Auto V gameplay posts. Because you put two tags together, alternatively if you don't want gameplay you can add -game tag with a minus sign in front of tag name instead and it will search for posts that depict GTA V instead of gameplay.

When searching you might want to look at what tags are available, click here. More about searching look in Tags in-depth section

Tag Explanation
purple_sky Search for posts with purple_sky tag
-winter Search for posts that don't have the winter tag
no_tags Search for posts that have no tags assigned
width:1920 Search for posts whose width is 1920 pixels
width:>1919 Search for posts whose width is above 1919 pixels
height:<1081 Search for posts whose height is below 1081 pixels
"hello world" Search for posts that have the words hello and world in text, order doesn't matter

Help: In-depth Tags

Here's a helpful table of available tags and attributes you can use to narrow down your searches.

Name Example Explanation
id id:1 Search for post whose unique ID is 1
author_id author_id:1 Post author's ID
op op:http://www.esmuralfs.lv/profile/1 Match an original poster's username or URL to profile
src src:http://www.esmuralfs.lv/view/1 Match original resource where post was taken from
directory directory:cute Search for posts in the "cute" directory
md5 md5:8aea73c6da8212327825ec017eae6a5d Match post with MD5 hash sum of 8aea73c6da8212327825ec017eae6a5d
ext ext:mp4 Match posts with file extension of mp4
mime mime:image/jpeg Match posts with MIME type of image/jpeg
size size:74925 Match posts with disk size in bytes of 74925
Note it is very hard to get this right so don't bother
height height:1440 Match posts whose height is 1440 pixels
width width:2560 Match posts whose width is 2560 pixels
category category:image Search for image posts
year year:2026 Search for posts uploaded in Year 2026
month month:10 Search for posts uploaded in October (tenth month)
day day:1 Search for posts uploaded on the first day of every month
score score:1 Look for posts whose score is 1 point

Help: Uploading

Great, you want to share with your own content, please be aware that we only host high quality memes/other content, if you upload a low quality meme, be warned that it may be removed. And please don't upload content with watermarks, and make sure to attribute the original poster when uploading. You have the tools to do the right thing so you should utilize them. If whenever in doubt you can reach out on our Discord server.

Help: NSFW

By default NSFW-related posts are hidden, if you wish to view NSFW and only NSFW posts then you can add nsfw tag. But please be aware of consequences that come with viewing NSFW posts
